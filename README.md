## Installation
This repository is developed and tested on

- Ubuntu 20.04
- Conda 4.9.2
- CUDA 11.4
- Python 3.7.11
- PyTorch 1.9.0

## Environment Setup
* Required environment is presented in the 'PID.yaml' file
* Clone this repo
```bash
~$ git clone https://github.com/sohyun-l/pid
~$ cd pid
~/pid$ conda env create --file PID.yaml
~/pid$ conda activate PID.yaml
```
